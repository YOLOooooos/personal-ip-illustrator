# Creator Personality

Use this before generating visual IP candidates. A personal IP must feel like the creator, not just the creator's topic.

## What To Extract

Extract only traits that can become visual rules:

- **Temperament**: calm, sharp, playful, skeptical, patient, restless, warm, austere.
- **Values**: craft, rigor, speed, independence, taste, evidence, generosity, defiance.
- **Recurring stance**: what the creator repeatedly pushes against or defends.
- **Voice**: plainspoken, poetic, analytical, ironic, tutorial, confrontational.
- **Visual taste**: sparse, tactile, technical, handmade, cinematic, dense, playful.
- **Anti-taste**: what would make the creator say "this is not me".
- **Reader feeling**: challenged, guided, reassured, provoked, equipped, amused.
- **Personal motifs**: tools, materials, rituals, symbols, scenes, colors, objects.

## Evidence Sources

Use any available evidence:

- user statements in the conversation
- profile text or bio
- representative posts or articles
- reference images
- existing projects
- explicit adjectives from the user

If evidence is missing, ask for 3-5 representative posts, a profile link, or five adjectives. If the user wants speed, infer cautiously and mark assumptions.

## Visual Translation

Translate personality into visual decisions:

| Personality Signal | Visual Translation |
| --- | --- |
| skeptical, anti-hype | narrow eyes/visor, diagnostic posture, red-pen or probe, restrained palette |
| patient craft | stitching, folding, measuring, repair tools, repeated marks |
| plainspoken rigor | simple geometry, no decorative clutter, strong contrast, visible labels only when useful |
| playful curiosity | elastic pose, odd companion object, warmer palette, surprise prop |
| sharp critique | angular silhouette, cutting tool, spotlight, contrast scene |
| generous teacher | guide posture, open hands, map, soft spacing |

## Candidate Requirements

Each candidate must answer:

- Which creator trait does this express?
- Which creator trait is missing or weak?
- What would make this candidate drift into a generic mascot?
- What prop, silhouette, posture, or material makes it recognizably personal?

## Fit Test

Before finalizing an IP, run this test:

1. Could another creator in the same niche use this unchanged?
2. Would the creator feel embarrassed to use it repeatedly?
3. Does it show a point of view, or only a topic?
4. Can it express at least three moods without losing identity?
5. Are the anti-tastes specific enough to prevent drift?

Revise the IP if the answer to 1 is yes, 2 is yes, 3 is no, 4 is no, or 5 is no.
